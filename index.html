使用示範
STEP 01
📸 拍照上傳藥品
在 LINE 對話中拍攝藥品標籤或藥盒，直接傳送圖片給 Bot。
STEP 02
🔍 OCR 辨識藥品名稱
後端收到圖片後，透過 OCR 引擎提取藥品名稱文字，自動解析關鍵字。
STEP 03
📍 回傳存放位置
比對藥品資料庫，即時回傳存放位置、藥架編號等資訊。

位置對照表
1. 使用者在 LINE 傳送藥品圖片
2. LINE Messaging API 發送 Webhook 事件到 GCP Cloud Run
3. Node.js 伺服器接收事件，下載圖片
4. OCR 引擎辨識圖片中的藥品名稱文字
5. 比對藥品資料庫，查詢存放位置
6. 透過 LINE Messaging API 回傳查詢結果
